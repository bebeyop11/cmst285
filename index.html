Project Concept

  • I'll be developing a multi page portfolio website that showcases the projects we've worked on so far.
  • This project is intended for future employers/clients.

Site Map

Web pages

  1. A Home page with a "Featured Projects" section,which will give a little description about the projects and the technologies used.
  2. An "About me page", which gives a little detail about my background and skills.
  3. A form that can be filled out by future clients.
  4. A "My works page which talks about the projects in more detail including a link to each project.
  5. "Contact me" page which will have my contact info, including a Mailto link for anyone trying to reach me.
